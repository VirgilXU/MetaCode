# Summary

- MetaCode 的目标不是积累越来越多的代码片段，而是沉淀可组合、可验证、可复用的能力单元。
- 一个能力元代码应该像软件能力世界里的 Token。它不一定独立构成完整工程，但可以和其他能力组合成工作流。
- V0.1 原型只验证身份证、Context、组合引擎和 Workflow 配置。暂时不接向量数据库，也不做自动 Agent。
