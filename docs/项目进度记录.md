## 下阶段建议

```text
从用户目标反推 workflow 的任务入口
```

优先任务：

## 分析时可重点关注

后续分析这个项目时，建议重点看五个问题：

1. MetaCode 的复用率是否持续上升。
2. 新 workflow 的生成成本是否下降。
3. 失败是否能被结构化定位，而不是靠人工阅读日志。
4. 能力图谱里的核心节点和桥接节点是否稳定。
5. 与传统 AI 编程相比，MetaCode 是否在中后期减少返工。
